# PSOC Control C3 Main Line (PSC3)

PSOC Control C3 Main Line (PSC3) is a microcontroller designed by Infineon, based on the Arm Cortex-M33 architecture.

## Notes

- This implementation is largly inspired by the Infineon Project generated by the ModusToolbox (MTB).
  By downloading the MTB, you can generate projects for the PSC3 which demonstrate how to use the various peripherals.
- During debugging some registers like TCPWM0 are only accessible when powered on and clocked.
